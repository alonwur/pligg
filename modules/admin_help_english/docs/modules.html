Pligg Modules

Pligg Modules are plugins that provide additional functionality to a Pligg site. They can perform simple tasks such as filtering swear words from all input fields to more advanced functions like providing an image upload applet to attach images to articles.

Downloading

You can download modules from the Pligg Forum Module Area. There are many free modules available for download on the forum, but you can also purchase "premium modules" from third party sites and the Pligg Pro Shop. Premium modules are ones that authors provide for a small fee to compensate them for their time spent working on it.

Installing and Upgrading

Installation is a simple 4 step process:

  1. Download and extract the module file from the internet
  2. Upload the extracted folder to your /modules directory where Pligg is installed
  3. Navigate to your Pligg Admin Panel and click on the Modules link. Then click install for the module(s) you want to install
  4. Configure your module. Usually documentation will be provided with the module with instructions
